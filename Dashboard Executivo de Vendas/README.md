# Dashboard Executivo de Vendas | Power BI

## Objetivo

Desenvolvimento de dashboard para acompanhamento dos principais indicadores comerciais, permitindo monitorar faturamento, cancelamentos, metas e desempenho da equipe de vendas.

## Tecnologias

- Power BI
- Power Query
- DAX
- Excel

## Indicadores

- Faturamento por ano
- Faturamento por vendedor
- Faturamento por forma de pagamento
- Cancelamentos
- Metas
- Evolução diária das vendas
- Histórico das vendas

## Funcionalidades

- Filtros por vendedor
- Filtros por período
- KPIs comerciais
- Ranking de vendedores
- Análise de metas
