
# Dashboard Executivo de Recursos Humanos

## Visão Geral

## Objetivo

Desenvolvimento de dashboard em Power BI para acompanhamento dos principais indicadores de Recursos Humanos.

## Tecnologias

- Power BI
- Power Query
- DAX
- Excel

## Indicadores

- Total de colaboradores
- Média salarial
- Média de idade
- Salário por departamento
- Salário por filial
- Faltas por filial
- Distribuição por gênero

## Principais funcionalidades

- Segmentação por departamento
- Segmentação por filial
- Filtros dinâmicos
- Navegação interativa
